## Setup

### Prerequisites

- Java `>=17`
- Gradle `>=9`
- Python `>=3`

*Python is not needed to run this app, but it is used in side `scripts/`

### Development

```sh
python .\scripts\run.py dev
```

### Build

```sh
python .\scripts\run.py build
```

The final JAR is indide `dist/`
